# Prism Design System 3.0

A unified design system for every endpoint experience. Built for consistency, accessibility, and speed across every product surface in Endpoint Central.

## Overview

Prism is the design system powering Endpoint Central — a comprehensive set of foundations, components, and AI-driven patterns crafted to help teams design, build, and scale consistent endpoint workflows. From design tokens to production-ready components, Prism gives teams everything needed to ship beautiful, accessible products fast.

## What's inside

- **Foundation** — Tokens, color, typography, spacing, grid, elevation, radius, border, iconography, illustrations, logos, content, accessibility, patterns
- **Components** — 60+ UI components organized by category (Forms & Inputs, Data Display, Navigation, Messaging & Feedback, Layout & Structure, Images & Icons)
- **AI Experience** — Zia AI patterns including Zia Input Field, Ask Zia chat, Zia Button, Zia Sidebar, Zia Search, Zia Tab, and Zia Error
- **Resources** — Supporting documentation and design assets

## Getting started

Open `index.html` in your browser to explore the design system. Each component, foundation, and AI pattern has its own dedicated page.

```bash
# Clone the repo
git clone https://github.com/kales-20817/Prism-design-system-3.0.git

# Open in browser
open index.html
```

## Features

- **Light & dark theme** support across every page
- **Interactive AI components** with live demos for Zia patterns
- **Responsive layouts** that scale across screen sizes
- **Token-driven** styling using CSS custom properties

## Tech stack

- HTML5
- CSS3 (custom properties, gradient borders, theming)
- Vanilla JavaScript for interactions
- Zoho Puvi font family
